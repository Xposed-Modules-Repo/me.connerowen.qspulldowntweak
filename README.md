# QS Pulldown Tweak

This is an extremely simple Xposed module that allows you to fully expand the quick settings UI with a single swipe down from either the left or right side of the status bar. I wrote this because it was the one feature from the [Dark Tricks Xposed module](https://github.com/darkeyes84/dark_tricks) that I missed, and that module hasn't been updated beyond Android 14. I ported the code for this from that module and modified it to make it work on Android 16+ (Android 15 might work too but is untested).

I don't necessarily have plans to port more features from the original Dark Tricks module to this one, but if there's another feature from that module that's desired then I'll implement it!

## Supported Devices

Android 15+ only. I've only tested this on my Motorola RAZR+ 2025 running Android 16, but this module should work regardless of Android skin (AOSP/OneUI/Oxygen/etc).
